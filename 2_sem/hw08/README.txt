НАУЧНЫЙ КАЛЬКУЛЯТОР

Возможности:
1. Выполняет операции сложения, вычитания, умножения, деления
2. Поддерживает числа с знаками после запятой
3. Есть кнопка очистить все
4. Меняет знак у числа
5. Выполняет тригонометрические функции
6. Вычисляет корень
7. Может удалить последний разряд числа

Известные баги, недочеты:
1. Не выполнены unit tests: тяжело придумать unit tests для кнопок
2. После взятия какой-либо функции от float числа,
результат с выполнением бинарных операций вычисляется некорректно
3. При многократном ( > 1 раза) нажатии кнопок с бинарными операциями калькулятор
выдает Exception 
4. Нельзя поменять знак у float числа, заданного пользователем

Ожидаемое время выполнения: 2 часа
Реальное время: 5 часов
